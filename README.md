# Physio SPA

Production-ready single-page app built with React, Vite, TypeScript, React Router v6, and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

## GitHub Pages deployment

This project is configured to deploy to GitHub Pages under `https://<USERNAME>.github.io/REPO_NAME/`.

1. Ensure the repository name matches the `REPO_NAME` placeholder.
2. Update the GitHub Pages settings to deploy from the `gh-pages` branch.
3. Push to the default branch to trigger the GitHub Actions workflow.

The workflow builds the site and publishes the `dist` folder to the `gh-pages` branch.
